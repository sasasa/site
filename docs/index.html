アシュタンガヨガとは

アシュタンガヨガの特徴

呼吸と動作をリンクさせて決まった順番にポーズをとります。 呼吸、ポーズの順番、視点などが決まっています 運動量が多くエネルギッシュなヨガです。 体を動かすのが好きな人におすすめです。 もちろん体を動かすのが苦手な人でも虜になるほど爽快感があります。当ヨガスタジオは 大濠公園のすぐそばにありアクセスしやすく毎朝練習することが出来ます。

アシュタンガヨガの効果

アシュタンガヨガの効果は数えきれません。 爽快感がありストレスを軽減する効果があります。自然と集中力も高まります。 運動量が多く筋力アップ→代謝アップ→脂肪燃焼という、うれしいダイエット効果もあります。 毎朝練習すれば効果もアップします。

クラスとは

「レッドクラス」はインストラクターの指導どうりにで全員が一斉に動くスタイルのクラスです。 「マイソールクラス」は、同じ場所に集合はしますが、自分のペースで個人練習をしてインストラクターが一人一人に声をかけていきます。 朝早くから練習をしますので、継続した練習をすることが比較的容易です。
